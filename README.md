<h1>Welcome to my project of implementation of OAuth 2.0 in Django Rest Framework.</h1>
<p>I am complete this Oauth by using django's built in package named django-allauth for social login in django and another builtin package named dj-rest-auth for RestAPI endpoints.</p>
<br>
<br>
<h2>for initiating this--></h2>
<p> first clone this repo in your system: </p>
<code>git clone https://github.com/mangalji/Oauth_implementation_2.git</code><br>
<p>second initiate the virtual environent: </p>
<code>python3 -m venv .venv</code><br>
<p>now, activate the virtual environent: </p>
<code>source .venv/bin/activate</code><br><br>
<p>third install the dependencies: </p>
<code>pip install -r requirements.txt</code><br>
<p>now run the app: </p>
<code>python3 manage.py runserver</code>
